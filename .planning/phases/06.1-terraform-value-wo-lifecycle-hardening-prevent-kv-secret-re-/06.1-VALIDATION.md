## Sampling Rate

- **After every commit touching `infra/**`, `scripts/**`, or the two workflow files**: Static TF CI job runs on PR open + every push to the PR branch.
- **Local dev loop** (wrapper changes only): `bash scripts/tests/test-tf-apply-prod.sh` before commit.
- **Before any prod `terraform apply`**: the wrapper itself (`scripts/tf-apply-prod.sh`) executes a missing-file guard and dual-var-file invocation — this is the in-line operational sample, not a test.
- **Max feedback latency**: ~30s end-to-end via PR CI (checkout + setup + static guards).
